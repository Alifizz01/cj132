Solar Array — blocks, bypass diodes & series wiring

Two blocks in series; each block is 4 strings in parallel, each string 4 cells in series. Every block has its own bypass diode (cathode at +, anode at − — so it blocks in normal operation). Click any cell to cycle healthy → shaded → failed/open.

Two lessons:
1 — Inside a block (parallel): failing one cell opens only its own string; the other 3 hold the block at voltage, so it survives on ~75% current.
2 — Between blocks (series): the same current flows through both, so the weaker block sets the array current. Only when a block is fully dead does its bypass diode open a detour (block ≈ −0.6 V) so the healthy block still delivers — without it, a dead block chokes the whole series and its starved cells hot-spot.
