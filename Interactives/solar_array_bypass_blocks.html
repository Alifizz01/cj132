Legend

A · Cell states

Healthy cell. String current 1.0.
Shaded cell. Throttles its whole string to 0.2 (min-Isc / weakest-cell rule).
Failed (open) cell. Breaks its string: string current 0. (Shown red as a hot-spot.)
Click a cell to cycle Healthy → Shaded → Failed. Shift-click (or right-click) to repair.

B · Current flow

Live conductor: current flowing (+ → −). Arrows = conventional direction; thickness ~ current.
No current: dead / open conductor (static, no flow).
Reduced current (e.g. 0.2) draws thinner and marches slower.

C · Bypass diode

CONDUCTING: its block is fully dead → current routes around the block (block ≈ 0 V). BYPASS ON
Blocking (normal): block is producing, diode carries no current. BYPASS OFF
Each block has its OWN bypass diode (anti-parallel: cathode at block +, anode at block −). It conducts only when ALL strings in that block are dead.
Anatomy: ARRAY + / ARRAY − = array terminals. INTER-BLOCK JUNCTION = Block 1 (−) tied to Block 2 (+), where the two series blocks meet and share one common current. Array voltage = V_block1 + V_block2; array current = the weakest non-bypassed block.
